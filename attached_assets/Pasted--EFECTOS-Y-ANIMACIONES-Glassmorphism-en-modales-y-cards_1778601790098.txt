  • Scale transform 1.03
  • Glow box-shadow con color del acento
  • Reveal de información adicional con slide-up

- Micro-animaciones en botones: ripple effect al hacer click
- Contador de cantidad con animación flip (como los marcadores de béisbol)
- Toast notifications con animación slide-in y barra de progreso

🖥️ DETALLES TÉCNICOS ADICIONALES
═══════════════════════════════════════
- CSS custom properties (variables) para theming dinámico
- Todos los colores con versión --color-name y --color-name-rgb para usar en rgba()
- Transiciones con cubic-bezier personalizados (no usar linear ni ease genérico)
  Ejemplo: cubic-bezier(0.34, 1.56, 0.64, 1) para spring effect
- Scroll personalizado (scrollbar-width: thin con colores del tema)
- Responsive con breakpoints: 768px tablet, 1024px desktop, 1440px wide
- Reduced motion media query para accesibilidad (@media prefers-reduced-motion)
- Z-index system con variables CSS organizadas por capas
- Focus-visible styles para accesibilidad de teclado
- Cursor personalizado en áreas interactivas (pointer con escala)
